hexo问题：

发表于2023-10-07|更新于2023-10-21|知识

hexo问题：

  1. Hexo+Github博客：执行 hexo d 时，提示连接Github的22端口超时

    1. 原因貌似是因为需要更新ssh。

    2. 问题为

    3. 解决方法：

      重新配置密匙：

      1. git bash 窗口依次输入:（注意：双引号里改为github用户名与邮箱）

        git config --global user.name "name"

        git config --global user.email "111@gmail.com"

      2. 生成新密匙

        ssh-keygen -t rsa -C "111@gmail.com"

      3. 遇到Overwrite (y/n)? 输入y，其余一直回车。

      4. 打开.ssh/id_rsa.pub文件，将密匙配置到github的·setting -> SSH and GPG keys，重新执行hexo d
