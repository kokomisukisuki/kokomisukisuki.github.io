jm学习

网址为tonquer/JMComic-qt: 禁漫天堂，18comic，使用qt实现的PC客户端，支持Windows，Linux，MacOS

学习时间在2023年11月5日，此时尚未失效。api可以服务。

官方文档教程jmcomic

一个非常值得学习的，注释设置的非常的清楚的，水平非常高的python程序库，十分建议学习。

模块依赖关系如下:

被依赖方 <—- 使用方

config <—- entity <—- toolkit <—- client <—- option <—- downloader

version = ‘2.3.17’

config

这些函数都是定义在common模块中的一些常用工具函数的封装。

  • field_cache: 封装了common模块中的field_cache函数,用于缓存字段值
  • default_jm_debug_logging: 封装日志打印函数
  • default_postman_constructor: 封装Postman对象构造函数
  • default_raise_exception_executor: 封装异常处理函数
  • system_proxy: 封装获取系统代理设置函数
  • str_to_list: 封装字符串转列表函数
  • JmcomicException: 定义一个Jmcomic异常类

目的是:

  1. 从common模块导入对应的函数
  2. 定义函数接口方便调用
  3. 隐藏common模块的具体实现
  4. 提供更友好的调用接口

这样其他模块就可以方便调用这些常用工具,而不需要直接导入common模块。

同时也实现了解耦,如果common模块函数改动了,这里的定义不用改。起到了封装和解耦的作用。
