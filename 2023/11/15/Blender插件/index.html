Blender插件

outline helper：

三渲二当中负责添加边线的插件，来源自【Blender】一看就会！三渲二全流程渲染篇。

node wranger：

插件功能（仅常用部分）

（1）快速导入贴图

按下Ctrl+shift+T，选中目标贴图（PBR），插件会帮助你快速创建贴图材质，再也不用一个一个拖了

（2）快速混合材质

比如我要将这2个纹理混合，那么选中他俩，按下Ctrl+Shift+右键，会出现绿色链条，松开鼠标即可快速混合

（3）快速布置贴图

贴图赋予在物体身上，想要改变贴图的方向，大小，改变投影方式，就必须用到矢量节点

选中纹理贴图，直接按下Ctrl+T 即可

（4）节点预览

按下 Crtl+Shift 左键快速预览任何一个节点

而且，按住 Crtl+Shift 再次左键，会从上往下依次预览不同输出点（图中的123）

（5）重加载图片

按下 Alt+R 重新载入所有图片

Bool Tool

安装插件后，按下Ctrl+Shift+B就可以调用布尔功能，功能十分清晰

分两部分：Auto Boolean & Brush Boolean

Auto的意思是马上应用布尔，不可修改；Brush的意思是创建对应的布尔修改器，可调整

用了它，摆脱在修改器面板吸来吸去的烦恼。

UV ToolKit ★★★

这是一个需要下载的插件，百度自行搜索

有了这个插件，我把Rizom UV从工作流中踢出去了

插件功能（仅常用部分）

（1)UV展平

做3d的就知道这个功能有多重要！Blender自带的UV展开约等于没用

治愈强迫症的优秀功能，而且让UV更合理，圆柱形体再也不怕了

（2）棋盘格材质预览

快速创建棋盘格材质，多个档位清晰度，快速又清楚，整个过程舒舒服服

不多说，谁用谁知道

（3）UV重排

和原生uv展开对比是秒杀的效果

（4）相似uv叠起

相见恨晚的好插件，必备，必备，必备

animation nodes

Installation Guide ：： 动画节点文档 (animation-nodes.com)

blender-mmd

Releases · UuuNyaa/blender_mmd_tools · GitHub ，一个正在更新的mmd导入blender的工具插件。

也具有官方文档，使用起来比较的得心应手。

mikumikurig（mmr）

MikuMikuRig是一款集生成控制器，自动导入动画，自动布料为一体的blender插件。 注意：Blender3.0需要安装Rigify修复补丁

一键为任何骨骼生成Rigify控制器

一键为生成的控制器导入任何FBX动画，BVH动画和VMD动画。导入的动画将作为NLA片段存在。

一键把MMD刚体物理转换为布料系统

一键美化MMD骨骼

rigify
