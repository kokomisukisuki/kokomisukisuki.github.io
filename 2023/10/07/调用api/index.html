调用api

API（Application Programming Interface ， 应用程序编程接口）是一些预先定义的函数，目的是提供应用程序与开发人员基于某软件或硬件得以访问一组例程的能力（提供获取数据的入口），而又无需访问源码或理解内部工作机制的细节。表现为是一种界面软件程序，有助于像用户界面（UI）一样与其他软件互动。

API为一个应用程序提供了一种相互作用的方式。例如，一个应用程序可以向另一个应用程序请求数据并获得数据作为回报。API（应用程序接口）组件也可以让一个应用程序的功能，第二个应用程序可以使用它。万维网联盟（W3C）将API定义为 “一套用于构建软件应用的子程序定义、协议和服务”。
