A free and efficient obfuscator for JavaScript (including support of ES2022). Make your code harder to copy and prevent people from stealing your work. This tool is a Web UI to the excellent (and open source) javascript-obfuscator@4.0.0 created by Timofey Kachalov.

jsjiami

JS反混淆还原工具

JS反混淆还原工具是一个免费的基于Web的在线工具。可用于删除常见类型的JS混淆代码。大多数形式的JS混淆代码都可以在使用在线JS反混淆还原工具进行删除。
JS反混淆还原工具可以帮助程序员解密可能由于某种原因而被混淆加密的JavaScript代码。加密后的JS代码通过使用更为复杂难懂的语法来混淆用户的思维。
通过还原工具可以让代码恢复为最初编写时的原始代码。混淆是将代码更改为更难读懂的编码格式的过程。反混淆还原则与此过程相反。

JSFuck

JSFuck is an esoteric and educational programming style based on the atomic parts of JavaScript. It uses only six different characters to write and execute code.
It does not depend on a browser, so you can even run it on Node.js.
Use the form below to convert your own script. Uncheck “eval source” to get back a plain string.
